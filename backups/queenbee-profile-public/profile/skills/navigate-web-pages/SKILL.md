name: navigate-web-pages
description: Skill for navigating web pages and handling redirects and login requirements.
---

## Steps
1. **Navigate to a URL**: Use the `browser_navigate` tool to go to the specified URL.
2. **Handle Redirects**: If redirected to a login page, inform the user and provide them the option to log in manually.
3. **Open Main Page**: If requested, navigate to the main page in a new tab, following similar handling for redirects.

## Use Cases
- Useful for accessing web pages that require user credentials or end up at login pages.
- Provides a workflow for navigating complex sites with potential redirect behavior.
