# Opportunity Scorecard

Use this quick rubric when comparing ideas.

## Scores

### Expected Value (1-5)
- **1** tiny upside or unclear value
- **3** meaningful upside with believable path
- **5** outsized upside relative to effort/capital

### Difficulty (1-5)
- **1** hard: new domain, heavy ops, high coordination
- **3** moderate: doable with focused work
- **5** easy: can validate quickly with existing tools/skills

### Capital Efficiency (1-5)
- **1** expensive to test
- **3** moderate spend required
- **5** cheap to validate

### Speed to First Dollar (1-5)
- **1** long horizon
- **3** medium horizon
- **5** can monetize fast

### Durability (1-5)
- **1** fragile / easily copied / short-lived
- **3** decent staying power
- **5** compounds through data, systems, brand, or distribution

## Quick decision heuristic

Prefer opportunities with:
- EV >= 4
- Difficulty >= 3
- Capital Efficiency >= 4
- Speed to First Dollar >= 3
- clear unfair advantage

## Fast filters

Kill or downgrade ideas if:
- the edge is just "AI is hot"
- the market is already saturated with better-funded players
- there is no route to customers
- the thesis depends on several unproven assumptions at once
- the user cannot run a cheap test soon
